دورة جافا للمبتدئين

دورة أساسيات لغة Java - للمبتدئين هل أنت مستعد لدخول عالم البرمجة بلغة تُستخدم في كل مكان؟ ابدأ رحلتك مع دورة Java الأساسية وكن جاهزًا للانطلاق

ماذا ستتعلم؟

  • خطوات تثبيت بيئة العمل (JDK + محرر الكود)
  • فهم المتغيرات وأنواع البيانات
  • جراء العمليات الحسابية
  • اجراء العمليات المنطقية
  • استخدام الجمل الشرطية if و switch
  • الزيادة القبلية والبعدية
  • النقصان القبلي والبعدي
  • الحلقات (while - for)

    • نظام الدورة

  • واجبات بعد كل محاضرة
  • شرح تفصيلي للواجب في المحاضرة التالية
